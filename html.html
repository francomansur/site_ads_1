HTML

Esta página apresenta conteúdos e exemplos sobre HTML, a linguagem de marcação usada para estruturar páginas na web. Aqui você encontrará explicações, dicas e boas práticas para criar documentos HTML bem organizados e acessíveis.

O que é HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar e organizar o conteúdo de páginas na web. Ele define elementos como títulos, parágrafos, imagens, links e listas, permitindo que navegadores interpretem e exibam essas informações de forma visual para o usuário.

Estrutura básica

  • !DOCTYPE html: informa ao navegador que o documento segue o padrão HTML5.
  • html lang="pt-br": elemento raiz, com atributo lang definindo o idioma da página.
  • head: contém metadados, título da página, configuração de responsividade (viewport) e links para arquivos externos como CSS.
  • body: onde fica todo o conteúdo visível:
    • header: área do cabeçalho, normalmente com logo, título ou menu.
    • main: parte principal do conteúdo da página.
      • .sections-wrapper: contêiner que agrupa todas as seções do site. Ele centraliza o conteúdo, define uma largura máxima e aplica um espaçamento uniforme (gap) entre as seções, garantindo consistência visual e facilitando a manutenção do layout.
      • section: blocos que separam o conteúdo por tema ou função (ex.: introdução, sobre, contato).
    • footer: rodapé da página, geralmente com direitos autorais, links úteis ou informações de contato.

Essa organização torna o HTML mais semântico, fácil de manter e de aplicar estilos no CSS, especialmente quando usamos o .sections-wrapper para controlar layout e espaçamentos de todas as seções de uma vez só. 



<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
    <link rel="stylesheet" href="css/style.css"> <!-- Linka o style.css com no HTML -->
</head>
<body>

    <header>
        <!-- Cabeçalho / Menu -->
    </header>

    <main class="class="sections-wrapper">
        <section>
            <!-- Conteúdo da primeira seção -->
        </section>

        <section>
            <!-- Conteúdo da segunda seção -->
        </section>
    </main>

    <footer>
        <!-- Rodapé -->
    </footer>

</body>
</html>

Principais tags

header: Cabeçalho da página ou seção, normalmente com logo, título ou menu.

nav: Área de navegação com links para outras páginas ou seções.

main: Conteúdo principal da página.

section: Bloco de conteúdo sobre um tema específico.

footer: Rodapé da página, geralmente com direitos autorais ou links extras.

h1 a h6: Títulos, indo do mais importante (h1) ao menos importante (h6).

p: Define um parágrafo de texto.
